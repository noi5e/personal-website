Hey, I'm Will! 👋🏽

I'm a trans software developer (he/him) from Oakland, California, who likes to build things with React.

Most recently, in 2021, I was selected by Outreachy to be an Open Source Intern with Public Lab. I wrote system tests and did a massive debugging of their website's commenting system. Then I rewrote the entire thing using React—preserving the Rails backend—and deployed it to production. Read more about it here!

When not in front of a computer screen, I enjoy biking around the neighborhood, writing, playing uncommon musical instruments—including the kutiyapi, an indigenous Pilipinx boat lute—and thinking about all things food.
